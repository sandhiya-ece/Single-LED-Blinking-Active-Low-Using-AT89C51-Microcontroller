# Single LED Blinking (Active Low) Using AT89C51 Microcontroller

## Description
This project demonstrates Single LED Blinking using Active Low logic on the AT89C51 (8051) microcontroller with Embedded C. The LED turns ON when the output pin is LOW and turns OFF when the output pin is HIGH. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- Single LED
- 330 Ω Resistor
- power 

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- active_low.c – Embedded C source code
- active_low.uvproj – Keil project file
- STARTUP.A51 – Startup file
- active_low.pdsprj – Proteus project file
- circuit.cng - proteus circuit diagram
